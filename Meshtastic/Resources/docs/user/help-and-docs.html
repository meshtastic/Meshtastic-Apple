Ask Chirpy

On iOS 26+, tap the sparkle icon in the top-right corner to ask Chirpy questions about Meshtastic. Chirpy uses on-device AI grounded in this documentation to answer your questions. When internet is available, it may also reference meshtastic.org sources.
